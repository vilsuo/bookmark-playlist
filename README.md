# Bookmark playlist
Import your bookmark html file and view it as a playlist

## How to start
### Starting server
Inside server folder
```
npm install
npm run dev
```
### Starting cliend
Inside client folder
```
npm install
npm run dev
```
After server and client are up, you can access the client in PORT 5173
